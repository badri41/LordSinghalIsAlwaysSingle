## 1. Introduction
This project aims to develop a web portal for exploring historical Air Quality Index (AQI) and pollution data across major Indian cities (2021–2026). The platform enables users to visualize trends and compare pollution levels. The system uses historical data (2021–2026) and focuses on visualization and analysis. Future AQI prediction may be incorporated using machine learning models.

## 2. System Overview
**Frontend**
* Built using React.js for dynamic UI
* Interactive charts and visualizations
* Responsive design

**Backend**
* Node.js and Express.js for API handling
* Data processing and request management

**Database**
* MySQL for structured storage
* Optimized queries for fast retrieval

## 6. Sample Analytical Queries

* Dates of extreme AQI spikes (Anomaly Detection)
* Monthly average PM2.5 levels
* Yearly AQI growth/decline rate
* Dominant pollutant (PM2.5 vs PM10)
* Worst month for air quality
* AQI patterns (weekdays vs weekends)
* Impact of rainfall on AQI
